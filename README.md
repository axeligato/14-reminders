14 Reminders from Me to You
A personalized anniversary gift webpage featuring 14 love notes - 2 for each year together.
About
This is a simple, elegant web page created to celebrate our 7-year anniversary. Each dropdown reveals a thoughtful quote from famous authors that responds to everyday moments we share together.
Features

Live countdown showing days, hours, and seconds since we met (February 14, 2019)
14 interactive dropdowns with partner quotes paired with literary responses
Minimalist Victorian-era inspired design with black background and elegant serif typography
Fully responsive for mobile and desktop viewing

How to View
Simply open the page in any modern web browser. Click on any question to reveal its answer.
Technical Details
Built with:

Pure HTML5
CSS3 (with glassmorphism effects)
Vanilla JavaScript (no dependencies)
Google Fonts (Playfair Display)
